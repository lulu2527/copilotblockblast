Ta grille

Tes 3 pièces

Remplis-les dans le même ordre que dans le jeu (A, B, C). Tu peux analyser avec 1, 2 ou 3 pièces.

Comment ça marche

  1. Tape les cases déjà remplies pour reproduire ta grille actuelle.
  2. Choisis tes 3 pièces du moment dans le tiroir (A, B, C).
  3. Appuie sur Analyser : l'algorithme teste tous les ordres et tous les emplacements possibles.
  4. Suis les étapes affichées sur ta partie réelle, puis valide pour préparer le tour suivant.

Priorité du calcul : lignes complétées immédiatement → maintien du combo (perdu après 3 coups d'affilée sans ligne) → espace gardé pour les grandes pièces → limitation des trous isolés. Le combo se conserve d'un tour à l'autre : valide tes coups pour qu'il soit suivi correctement.

Choisis tes pièces

Chaque pièce choisie remplit le slot surligné, puis passe au suivant. Tape un slot pour le corriger.
